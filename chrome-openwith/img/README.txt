图标文件说明
===========

本目录需要包含以下图标文件：

1. icon16.png (16x16像素)
2. icon19.png (19x19像素)
3. icon38.png (38x38像素)
4. icon48.png (48x48像素)
5. icon128.png (128x128像素)

您可以通过以下方式创建这些图标：

1. 使用图像编辑软件（如Photoshop、GIMP等）创建相应尺寸的PNG图标
2. 使用在线图标生成工具
3. 打开项目根目录中的icon-generator.html文件，按照页面上的说明创建图标

图标设计建议：
- 使用简单、清晰的设计
- 确保在小尺寸下仍然可识别
- 使用与扩展功能相关的图形元素
- 可以使用字母"O"或"E"（代表"Open External"）作为图标的主要元素
- 建议使用蓝色或绿色作为主色调

临时图标解决方案：
如果您暂时没有图标，可以使用纯色背景的文字作为临时图标。例如，在蓝色背景上显示白色的"O"字母。
